style 행운의 카드 뒤집기란?

다양한 테마의 카드를 뒤집어 오늘의 운세를 확인하거나, 순서를 정하고, 벌칙을 결정할 수 있는 도구입니다. 직관적인 디자인과 실시간 언어 전환 기능을 통해 언제 어디서나 즐기실 수 있습니다.

tune 4가지 게임 모드

format_list_numbered 1. 순서 (숫자)

발표 순서나 등수를 정할 때 사용하세요. 숫자가 무작위로 숨겨집니다.

edit 2. 직접 입력

벌칙이나 경품을 직접 입력하여 술자리 게임이나 집안일 정하기에 활용해보세요.

stars 3. 아이콘 점괘 (한 장 제한)

오늘의 운세를 상징하는 아이콘과 상세한 메시지를 확인하세요. 직감을 믿고 단 한 장만 고를 수 있습니다.

psychology 4. 심리/조언 (한 장 제한)

결정이 필요하거나 마음의 위로가 필요할 때 사용하세요. 당신에게 꼭 필요한 특별한 메시지를 전합니다.

tips_and_updates 핵심 기능 안내

  • 실시간 언어 전환: 메뉴에서 한국어와 영어를 전환할 수 있습니다. 카드 뒷면의 아이콘과 결과 메시지가 선택한 언어에 맞춰 실시간으로 동기화됩니다.
  • 선택 제한 시스템: 점괘나 조언 모드에서는 신중한 선택을 위해 카드를 한 장만 뒤집을 수 있습니다. 카드를 한 번 뒤집으면 안내 문구가 자동으로 사라집니다.
  • 반응형 디자인: 모바일 환경에 최적화되어 터치만으로 간편하게 모든 기능을 이용할 수 있습니다.
